# Ex.No:3(E) INNER CLASS

## QUESTION:
Write a Java program to create an inner class and access it from the outer class.
## AIM:
To use an inner class and access its method from outer.
## ALGORITHM :
1.	Start the program.
2.	Import the necessary package 'java.util'
3.	Read the user’s name as input using the Scanner class in main.
4. Create an OuterClass object and call a method that uses the inner class.
5. Inside the outer method, create an InnerClass object and call its displayMessage.
6. Print the greeting message generated by the inner class using the given name.

 ```

```

## SOURCE CODE:
```
import java.util.*;
public class OuterClass {
    class InnerClass {
        void displayMessage(String name) {
            System.out.println("Hello, " + name + "! This message is from the Inner Class.");
        }
    }

    public void callInner(String name) {
        InnerClass inner = new InnerClass();
        inner.displayMessage(name);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String inputName = sc.nextLine();
        OuterClass outer = new OuterClass();
        outer.callInner(inputName);
    }
}

```
